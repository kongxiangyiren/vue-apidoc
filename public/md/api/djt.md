接口地址 `https://api.kongxiangyiren.top/api/djt`

输出类型 `JSON`

请求方式 `GET`

请求示例 `https://api.kongxiangyiren.top/api/djt`

## 返回参数

| 参数 | 类型   | 说明                        |
| ---- | ------ | --------------------------- |
| code | number | 状态码,0 为成功，其他为错误 |
| msg  | string | 状态信息                    |
| data | string | 美味鸡汤                    |

## 返回示例

```json
{
  "code": 0,
  "msg": "ok",
  "data": "用美颜手机自拍多了，越来越不知道，自己有多丑了。"
}
```
